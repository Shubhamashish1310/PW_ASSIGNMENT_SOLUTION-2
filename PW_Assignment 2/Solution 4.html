LIST OF HTML TAGS

  1. <!DOCTYPE html :;> This tag is used to declare the document type.
  2. <head> Contain information about the HTML document, such as the title and meta tags.
  3. <html> This tag is used to define the beginning of an HTML document.
  4. <h1> to <h6> Define headings of different sizes.
  5. <img> This tag is used to insert an image.
  6. <p> Defines a paragraph of text
  7. <table> ,<tr>,<td>,<th>: Create tables
  8. <hr> Inserts an image.
  9. <a> for link
  10. <b> make bold text
  11. <body> main HTML part
  12. <br> for break
  13. <div> it is a division or part of an HTML document
  14. <i> to make an italic text
